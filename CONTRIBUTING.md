# Contributing

Cảm ơn bạn quan tâm đến dự án này!

## Code Style

- Sử dụng Vanilla JavaScript (không dùng framework)
- Comments bằng tiếng Việt
- Mỗi file JS có header comment giải thích chức năng
- Error handling cho tất cả API calls
- Loading states cho async operations

## Cấu Trúc Code

- `js/firebase-config.js`: Firebase configuration
- `js/auth.js`: Authentication system
- `js/game-engine.js`: 3D game engine với Three.js
- `js/gamification.js`: Hệ thống points, levels, achievements
- `js/ai-counselor.js`: AI counselor với Gemini
- `js/chat-system.js`: Real-time chat
- `js/main.js`: Main file kết nối tất cả modules
- `js/mini-games/`: Các mini-games

## Testing

Trước khi submit PR, hãy test:
- Authentication flow
- Tất cả mini-games
- AI counselor
- Chat system
- Mobile responsive

## Pull Request Process

1. Fork repository
2. Tạo feature branch
3. Commit changes
4. Push to branch
5. Create Pull Request

## Questions?

Nếu có câu hỏi, hãy tạo issue trên GitHub.
